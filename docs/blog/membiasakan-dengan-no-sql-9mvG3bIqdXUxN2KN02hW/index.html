Membiasakan dengan No-SQL Database (Firebase)

Apakah benar Firebase terbukti bisa membantu?

 

Firebase sebagai salah satu MBaaS (Mobile Backend as a Service) mengalami perubahan besar-besaran pada May 2016, dengan versi terakhirnya dinamakan Firebase 3.0. Dibandingkan sebelumnya yang mana fungsi Firebase terbatas pada authentication service dan real-time database saja, kali ini Firebase memiliki hampir seluruh fungsi yang dibutuhkan sebagai backend service dari Mobile Application.

 

Diwaktu yang sama dengan peluncuran Firebase 3.0 ini, saya membuat web portfolio ini menggunakan firebase dimana data portfolio serta quotes dan blog meggunakan firebase firestore. Awal perubahan dari SQL mode ke No SQL memang cukup sulit namun dalam penerapannya dengan banyak belajar serta implementasi maka menggunakan No SQL database ternyata cukup mudah.

 

Firebase realtime mode menjadi fitur unggulan dimana saya cukup banyak menggunakannya untuk membuat chat realtime serta aplikasi realtime lainnya. jadi ayo migrasi ke No SQL Database  wkkwkw :)
