Astro framework sangar tapi belum dikenal orang

Astro adalah framework baru yang dirancang untuk kecepatan. Dan astro melakukannya dengan sempurna. Astro dapat berintegrasi dengan kerangka kerja favorit developer seperti React, Vue, dan Svelte untuk membuat halaman yang dirender ke HTML untuk memberikan kecepatan pemuatan yang gaada obat.

 

Tidak hanya cocok dengan jumlah elemen interaktif yang rendah, tetapi bahkan dengan situs yang sangat interaktif, ini dapat mengungguli beberapa framework web utamnaya js yang paling populer.

 

Apakah ada kekurangan? Tentu saja, ada. Tetapi untuk saat ini, Astro terfokus untuk tetap positif dan memeriksa cara membangun situs sederhana dengan kerangka kerja yang kuat dan secepat kilat ini.

 

Setup 👷‍♂️

 

Memulai proyek Astro baru semudah dengan kerangka kerja modern lainnya. Ini menyediakan alat CLI sederhana untuk mengatur situs pertama Anda.

 

npm create astro@latest

 

Saat menjalankan perintah, CLI akan meminta beberapa opsi kepada Anda. Sebagai permulaan, ia akan menanyakan di mana harus meletakkan file proyek baru, template apa yang digunakan (saya menggunakan "Hanya dasar-dasar"), jika Anda ingin menginstal dependensi NPM (ya), jika Anda ingin mengatur Git (saya melakukannya), dan akhirnya, jika Anda ingin menggunakan TypeScript yang saya katakan "Sangat (disarankan)".

 

As you can see, everything we need has been generated and we can now run the project locally by running

 

npm run dev

 

in the terminal.

 

Be sure to navigate inside the new project folder!
