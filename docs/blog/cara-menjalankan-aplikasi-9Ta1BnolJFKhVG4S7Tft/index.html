Cara menjalankan aplikasi Flutter di device tanpa USB

Menjalankan aplikasi flutter pada real device biasanya menggunakan kabel USB sebagai media penghubung antara komputer dan android device, dalam tutorial ini akan menunjukan cara agar deployment bisa berjalan pada real devices namun tanpa menggunakan kabel USB.

 

ðŸ›  Persiapan

 

  • USB kabel (lho kok pake USB kabel, tenang cuman sekali doang kok)
  • Smartphone / Android Devices
  • Komputer
  • Kopi biar santuy

 

Pastikan komputer anda sudah terinstall flutter dan adb dan pernah melakukan deployment pada real devices sebelumnya. Tutorial ini hanya berlaku bagi anda yang telah terbiasa melakukan development mobile apps menggunakan flutter.

 

ðŸ’¡ Langkah - Langkah

 

  1. Tancapkan USB kabel ke komputer dan juga smartphone
  2. Pastikan komputer dan smartphone anda berada dalam satu jaringan
  3. Pastikan anda telah menyalakan USB Debugging di smartphone
  4. Buka command prompt dengan cara (Win + R), lalu ketik "cmd", lalu enter
  5. Ketik command berikut adb tcpip 5555
  6. Cek ip address pada smartphone android anda
  7. Lepaskan kabel USB dari komputer dan smartphone
  8. Ketik command berikut untuk menghubungkan koneksi adb ke smartphone adb connect <smartphone_ip_address>:5555
  9. Selamat smartphone dan komputer anda telah terhubung. Anda dapat melakukan deployment seperti biasa
